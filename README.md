# Internship-Project-2
Traffic Sign Recognition Using CNN

Tech Stack: Python, PyTorch, TorchVision, OpenCV, Streamlit, Pandas, NumPy, Matplotlib, Pillow
Model: Custom CNN (Convolutional Neural Network)
Dataset: GTSRB (German Traffic Sign Recognition Benchmark)
Performance: 97.28% Official Test Accuracy
Deployment: Interactive Streamlit Dashboard for Traffic Sign Recognition.

Used:
Batch Normalization

ReLU Activation

Max Pooling

Dropout (0.5)

Cross Entropy Loss

Adam Optimizer

GPU Training (RTX 2050)
